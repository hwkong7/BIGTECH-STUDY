# 컴퓨터가 이해하는 정보

컴퓨터는 자바, C++, 파이썬과 같은 프로그래밍 언어를 직접 이해하지 못한다

- 데이터와 명령어
  - 컴퓨터가 이해할 수 있는 정보

## 데이터

- 숫자, 문자, 이미지, 동영상 ⇒ 정적인 정보
- 컴퓨터와 주고받는 정보나 컴퓨터에 저장된 정보 자체
- ex) 1, 2, “hello world”, ‘cat.jpg’
- 있는 그대로의 정보

## 명령어

- 수행할 동작 | 수행할 대상
- 이 데이터를 활용하는 정보

⇒ 데이터는 명령어에 **종속적인 정보**이며, **명령의 대상**이자, **명령어의 재료**

⇒ 명령어를 실행하는 주체는 컴퓨터의 핵심 부품 중 하나인 **CPU**

- CPU의 종류에 따라 실행 가능한 세부적인 명령어의 종류와 처리 양상이 달라질 수 있다.

# 컴퓨터의 핵심 부품

CPU

# CPU(Central Processing Unit)

정보를 읽어 들이고, 해석하고, 실행하는 부품

![image.png](attachment:fa959cf9-ec5b-4403-87f3-b7b8003e22a3:image.png)

## 산술논리연산장치(ALU, Arithmetic and Logic Unit)

- 사칙 연산, 논리 연산과 같은 연산을 수행할 회로로 구성되어 있는 일종의 계산기
- CPU가 처리할 명령어를 실질적으로 연산하는 요소

## 제어장치(CU, Control Unit)

- 명령어를 해석해 제어 신호라는 전기 신호를 내보내는 장치
  - 제어 신호 : 부품을 작동시키기 위한 신호
- 제어 신호 → 메모리 ⇒ 메모리를 작동시킬 수 있고,
- 제어 신호 → 입출력장치를 ⇒ 입출력장치를 작동시킬 수 있다.

## 레지스터(register) ⭐

- CPU 내부의 작은 임시 저장장치로, 데이터와 명령어를 처리하는 과정의 중간값을 저장한다.
- CPU 내에는 여러 개의 레지스터가 존재하며 각기 다른 이름과 역할을 가지고 있다.

메모리

# 메모리(RAM)

- : **실행 중인** 프로그램을 구성하는 데이터와 명령어를 저장하는 부품
- 메인 메모리(main memory) 역할
  - 하드웨어 - RAM과 ROM
- CPU가 읽어 들이고, 해석하고, 실행하는 모든 정보는 어딘가에 저장되어 있어야 하며, 이 정보를 저장하는 장치가 바로 메모리

## 주소

- : CPU가 메모리에 접근할 때 컴퓨터가 빠르게 작동하기 위해서는 메모리 속 데이터와 명령어가 정돈되어 있기 위해 사용되는 개념(원하는 정보로 접근하기 위해서)

![image.png](attachment:418b430f-c181-45c1-835f-952214a63e17:image.png)

## 휘발성(volatile)

- 전원이 공급되지 않을 때 저장하고 있는 정보가 지워지는 특성

캐시 메모리

# 캐시 메모리(cache memory)

- CPU가 조금이라도 더 빨리 메모리에 저장된 값에 접근하기 위해 사용하는 저장장치
- 빠른 메모리 접근을 보조하는 저장장치인 셈
- CPU 안에 위치하기도 하고, CPU 밖에 위치하기도 하며, 여러 종류 있다.

![image.png](attachment:15110e35-d990-4628-8c65-994dbd59d1f2:image.png)

보조기억장치

# 보조기억장치

- 전원이 꺼져도 데이터가 사라지지 않고, 프로그램이나 파일 등을 오랫동안 저장해 두는 장치

![image.png](attachment:f98aab0b-d1a1-410f-876a-e100068af166:4427db80-f664-4ea2-b0bf-37c05075057a.png)

- **비휘발성(non-volatile)**
  - 전원이 꺼져도 저장된 정보가 사라지지 않는 성질
- **보관할** 프로그램을 저장한다.

유의할 점

- CPU가 보조기억장치에 저장된 프로그램을 곧장 가져와 실행할 수 없다.
- 어떠한 프로그램 실행 ⇒ 보조기억장치에서 보관하고 있는 프로그램을 메모리로 복사

입출력장치

# 입출력장치(input/output device)

- 컴퓨터 외부에 연결되어 컴퓨터 내부와 정보를 교환하는 장치
- 컴퓨터에 어떠한 입력을 할 때 사용하는 장치가 입력장치, 컴퓨터의 정보를 받기 위해 사용하는 장치 ⇒ 출력 장치

![image.png](attachment:9c66a8cb-1468-4877-bfca-3ad3b41bd3d5:image.png)
