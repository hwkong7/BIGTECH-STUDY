# 명령어

- 수행할 동작과 수행할 대상
- 연산 코드
    - 명령어가 수행할 동작
- 오퍼랜드
    - 사용될 데이터가 저장된 위치

## 기계어와 어셈블리어

- 기계어
    - 0과 1로 표현된 정보를 있는 그대로 표현한 언어
    - 어떤 프로그램인지 짐작하기 어려움
- 어셈브리어
    - 기계어를 읽기 편한 형태로 단순 번역
    - CPU가 이해할 수 있는 명령어의 종류와 동작을 알 수 있음

## 명령어 사이클

- 인출 사이클
    - 메모리에 있는 명령어를 CPU로 가져오는 단계
- 실행 사이클
    - CPU로 가져온 명령어를 실행하는 단계
- 간접 사이클
    - 명렁어를 실행하기 위해 한 번 더 접근하는 단계
